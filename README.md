# INDUSNET-B2B-G2B-Talent-Sourcing-Platform
INDUSNET is a B2B/G2B talent sourcing web platform connecting industry HRs with public &amp; social orgs to solve labor shortages. I led its end-to-end development, authored BRD/PRDs, defined the UX/UI, established partner criteria, and drove its Go-to-Market strategy, launching it via webinars to drive rapid adoption.
